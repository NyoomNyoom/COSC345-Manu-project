public final Integer questionTypeToInt(QuestionType questionType)
Content copied to clipboard

Converts a QuestionType value to an Int value according to the following map: QuestionType.PHOTO 0 QuestionType.SOUND 1 QuestionType.ENGLISH 2 QuestionType.MAORI 3 Other -1 Other includes QuestionType.ALL because translating that value is not a purpose of this class.

Return

An Int representing the QuestionType passed to this function. Please read the method description for special cases.

Parameters

questionType

The QuestionType value which this function will read and convert to an Int.
