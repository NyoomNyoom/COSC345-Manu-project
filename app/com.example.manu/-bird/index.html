A class which holds all the values needed to keep track of a bird for the given question. It takes in a random number from BirdAdapter.kt and then searches through a list of birds to find that number and then checks if the bird is valid for the quiz, and if it is, will update the values of the bird so we have all the needed values that were stored in the information list.
